## Build and run

*if you want to check this project, you must download and run the API.*

1. Download the source code
```
git clone https://github.com/Redooz/F1-2022-Races-API
```
2. Run the npm script to start
```
npm start
```
3. Go to the website
```
http://localhost:3001/
```
